O RAVI teve início como trabalho de conclusão de curso (TCC) de Caio Arantes, desenvolvido sob orientação do Prof. Dr. Lucas dos Rios Amaral. Atualmente, o RAVI é um projeto de código aberto, mantido com o apoio da FARM Analytica, empresa cofundada por Caio Arantes. O projeto mantém seu compromisso com a difusão tecnológica e com o espírito do software livre, contando com manutenção contínua e a implementação periódica de novas funcionalidades.

O que é o RAVI?

O Plugin RAVI (Remote Analysis of Vegetation Indices) é uma ferramenta desenvolvida para o QGIS que facilita o cálculo e a análise de índices de vegetação utilizando dados do Google Earth Engine (GEE). Ele permite o processamento de imagens de satélite para o monitoramento da saúde da vegetação, a avaliação da produtividade agrícola e a análise de mudanças ambientais, de forma integrada ao ambiente do QGIS.

Requisitos para uso

  • Uma conta ativa no Google Earth Engine
    Conta registrada e aprovada no GEE, que fornece acesso aos conjuntos de dados, APIs e ferramentas de processamento.
  • Um projeto no Google Cloud (com ID do projeto)
    Projeto do Google Cloud vinculado à conta do Earth Engine, utilizado para autenticação, habilitação das APIs e gerenciamento de cotas de uso.
