about=AGLgis is a graphical interface for Sentinel-1 SAR Backscatter Analysis-Ready-Data preparation in Google Earth Engine, inside QGIS. Build VV/VH backscatter time series for cloud-independent monitoring of crops, water and land surfaces, with border-noise correction, terrain flattening and speckle filtering. Requires a Google Earth Engine account (sign-in and a project ID are set up inside the plugin). / O AGLgis é uma interface gráfica para o preparo de dados de retroespalhamento SAR Sentinel-1 (Analysis-Ready Data) no Google Earth Engine, dentro do QGIS. Gere séries temporais VV/VH para monitoramento independente de nuvens de culturas, água e superfícies, com correção de ruído de borda, achatamento de terreno e filtragem de speckle. Requer uma conta do Google Earth Engine (login e ID de projeto são configurados dentro do plugin).

changelog=
    9.0 - RAVI plot date-range filtering: two-handle slider below the plot lets
          you narrow the cached time series by date span. Dragging updates the
          label live and re-renders the plot after settling (200 ms debounce).
          The date selection ANDs with existing filters (thresholds, manual
          per-date selection) and automatically constrains composite generation,
          batch download, CSV export, and single-image dropdown. RangeSlider
          widget gains optional label_fn for custom value→text mapping (e.g.
          day offsets to ISO dates), adaptive label width, and set_span()
          reconfig without signal emission. Numeric year-range path (MapBiomas
          transition, SYSI filters) untouched. AOI-area label on Inputs tab
          shows hectares below the layer picker. Management Zones module:
          integrated fork of Precision Zones for defining and analyzing
          management boundaries within fields, with zoning workflows, per-zone
          statistics, and export to QGIS for further analysis.
    8.0 - now built from the unified FARM tools codebase. Ships defaulting to this module; the other FARM tools modules are discoverable from the welcome page's "More tools" section and as separate plugins.
