about=AGLgis is a graphical interface for Sentinel-1 SAR Backscatter Analysis-Ready-Data preparation in Google Earth Engine, inside QGIS. Build VV/VH backscatter time series for cloud-independent monitoring of crops, water and land surfaces, with border-noise correction, terrain flattening and speckle filtering. Requires a free Google Earth Engine account (sign-in and a project ID are set up inside the plugin). / O AGLgis é uma interface gráfica para o preparo de dados de retroespalhamento SAR Sentinel-1 (Analysis-Ready Data) no Google Earth Engine, dentro do QGIS. Gere séries temporais VV/VH para monitoramento independente de nuvens de culturas, água e superfícies, com correção de ruído de borda, achatamento de terreno e filtragem de speckle. Requer uma conta gratuita do Google Earth Engine (login e ID de projeto são configurados dentro do plugin).
